## 电子书城（静态前端页面）

本项目为课程《互联网应用开发技术》课程大作业的**前端静态页面**实现，包含：

- 登录页：`pages/login.html`
- 主页（书籍列表）：`pages/books.html`
- 书籍详情：`pages/book-detail.html`
- 购物车：`pages/cart.html`
- 订单页：`pages/orders.html`

### 运行方式

直接用浏览器打开任意页面即可（建议从登录页开始）：

- `pages/login.html`

> 页面仅实现 UI 与跳转链接，不包含真实登录、下单等业务逻辑。

### 目录结构

```
assets/        静态资源（SVG 占位图）
css/           外部样式（无内联样式）
pages/         各功能页面（HTML5 语义化结构）
Ans.docx       前端主要使用Cursor生成，对话记录见此文档
```
